# DQ_propella

Data quality experiments using Propella annotations and linked FinePDFs metadata.

## Current focus

- Language-confidence feature extraction for FinePDFs/Propella document IDs.
- Small human-test samples for comparing raw text excerpts and Propella one-sentence descriptions.

## Notes

Large/generated artifacts are ignored by git. Keep raw data, model binaries, and processed CSV/Parquet outputs out of the repository unless explicitly needed.
